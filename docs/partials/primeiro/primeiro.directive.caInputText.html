caInputText
directive in module primeiro

Description

Componente padrão de input text

Usage

as element:
<ca-input-text
       label="{string}"
       colspan="{number}"
       ng-model="{object}"
       ng-required="{boolean}">
</ca-input-text>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
labelstring

Texto a ser exibido na parte superior do input

colspannumber

Quantidade de colunas que o input ocupara. Default: 3

ng-modelobject

Referência do input

ng-requiredboolean

Informa se o campo é obrigatório ou não. Default:false
